# 🎲 Python Dice Roller

A simple Dice Roller game built using Python.

## Features
- Roll two random dice
- Display both dice values
- Show the total of the dice
- Count the total number of rolls
- Display the highest dice sum
- Display the lowest dice sum
- Display the average dice sum
- Exit the game by pressing N

## Technologies Used
- Python 3
- random module

## How to Run

```bash
python dice_roller.py
```
